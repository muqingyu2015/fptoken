P1 — 行为风险

ID位置现象建议

BUG-103 FpGroupDataRebuild#flushto del_term_docid 未使用；reuse_bytes 固定 1024、dataId 前缀 TODO。 与 Lucene 写路径对照验收。

  • BUG-201 FpTokenBlockLevelPolicy 仅返回级别 1 或 3，注释中的「级别 2」未出现。
  • BUG-202 FpGroupDataRebuild#rebuildCommonTermToOrderFromHotDocs Javadoc 仍写 hotTermToOrder。
  • BUG-203 设计文档 FPBlockTreeTermsWriter 与源码大小写混用，集成检索需注意。
  • BUG-204 仍缺 FpGroupHotNgramBitIndex round-trip、FpTokenBlockOrchestrator 双路径集成测试（可在 LXDB 写段环境补）。
